# Embalor (PHP) - Arabic Packaging Growth Platform

منصة احترافية لإدارة طلبات الأومبلاج كخدمة ترفع المبيعات وتدعم العلامات التجارية.

## تحديثات مهمة
- واجهة أحدث + أماكن صور في الصفحة الرئيسية.
- شرح واضح لمسار 3D: العميل يرفع التصميم -> معاينة 3D -> تأكيد -> تصنيع.
- أقل كمية إنتاج: **50 حبة**.
- إشعارات رسائل جديدة (Notification Badge) مع تحديث تلقائي.
- الأدمن يمكنه رفع **Logo** للموقع بدل النص.
- التحكم في ألوان ونصوص الموقع وBase URL.

## تشغيل
1. افتح `http://your-domain/install/`
2. أكمل إعدادات DB + الأدمن.
3. بعدها استعمل `http://your-domain/` مباشرة.
