# Embalor (PHP) - Arabic Packaging Growth Platform

منصة احترافية لإدارة طلبات الأومبلاج (التغليف) كأداة **رفع مبيعات** وهوية بصرية قوية للمنتج.

## الجديد في هذا الإصدار

- ديزاين عصري بالكامل Responsive للهاتف والتابلت والكمبيوتر.
- إمكانية تحكم الأدمن في:
  - ألوان الموقع
  - عنوان ونص الواجهة
  - بيانات التواصل
  - `base_url` لفتح الموقع من `domain.com` مباشرة بدون `/public`
- إضافة **سعر تقديري** لكل طلب يمكن تعديله من الفريق.
- إضافة **شات داخلي داخل كل طلب** للتواصل بين العميل وفريق الشركة.

## الأدوار

- Customer: إنشاء ومتابعة الطلبات
- Worker/Designer: تنفيذ الطلبات + تحديث الحالة + السعر
- Admin: إدارة كاملة + إعدادات الموقع + إدارة المستخدمين

## التثبيت

1. افتح `http://your-domain/install/`
2. أدخل بيانات قاعدة البيانات + بيانات المدير + base URL.
3. بعد التثبيت افتح الموقع من `http://your-domain/` مباشرة.

## ملاحظات إنتاج

- اجعل `public/uploads` قابلاً للكتابة.
- احذف مجلد `install` بعد الإطلاق.
