# Embalor (PHP) - Arabic Packaging Growth Platform

منصة احترافية لإدارة طلبات الأومبلاج كخدمة ترفع المبيعات وتدعم العلامات التجارية.

## تحديثات مهمة
- إلغاء فكرة 3D وتعويضها بمسار واقعي: العميل يرسل طلبه -> نصنع شونتيو حقيقي -> موافقة -> إنتاج الكمية.
- الأدمن يقدر يزيد/يحذف أنواع الأومبلاج التي تظهر للعميل.
- الأدمن يقدر يحظر/يفك الحظر عن العملاء.
- رقم الهاتف صار إجباري في التسجيل.
- زر واتساب مباشر للموظفين للتواصل مع العميل من صفحة الطلب.
- علامة توثيق ✔ بجانب رسائل فريق الشركة (admin/worker/designer).
- تحسين الجداول في الهاتف عبر table responsive.

## تشغيل
1. افتح `http://your-domain/install/`
2. أكمل إعدادات DB + الأدمن.
3. بعدها استعمل `http://your-domain/` مباشرة.
