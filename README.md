# Embalor (PHP)

منصة عربية لإدارة طلبات تصميم وإنشاء الأومبلاج (التغليف) بخيارين رئيسيين:

1. **رفع تصميم العميل وتحويله إلى نموذج 3D للأومبلاج** مع اختيار الشكل والمقاسات.
2. **طلب تواصل مع الفريق** ليقوم مصممو الشركة باقتراح تصميم مناسب.

## المميزات

- واجهة عربية RTL بتصميم عصري.
- نظام مستخدمين وصلاحيات متعددة:
  - عميل (Customer)
  - مصمم (Designer)
  - عامل/موظف (Worker)
  - مدير (Admin)
- تسجيل دخول/تسجيل حساب.
- لوحة تحكم حسب الدور.
- إدارة الطلبات، الحالات، وإسناد الطلبات من لوحة المدير.
- رفع ملفات مرجعية للطلبات.
- مثبت (Installer) لإنشاء الإعدادات وقاعدة البيانات تلقائيًا.

## المتطلبات

- PHP 8.1+
- امتداد `pdo_mysql`
- MySQL 8+ أو MariaDB حديثة
- خادم ويب (Apache/Nginx) أو تشغيل محلي عبر PHP built-in server

## التشغيل السريع

1. انسخ المشروع إلى السيرفر.
2. اجعل مجلد `public/uploads` قابلاً للكتابة.
3. افتح:
   - `http://your-domain/install/`
4. أدخل بيانات قاعدة البيانات + بيانات حساب المدير.
5. بعد نجاح التثبيت:
   - `http://your-domain/public/`

## تشغيل محلي (تجريبي)

```bash
php -S localhost:8080 -t public
```

ثم افتح `http://localhost:8080`.

> ملاحظة: في بيئة التطوير، يمكن إبقاء المثبت. في بيئة الإنتاج يفضّل حذف مجلد `install` بعد اكتمال الإعداد.
